# Key to the Keys
- ⌘ : Command key
- ⌃ : Control key
- ⌫ : Delete key
- ← : Left arrow key
- → : Right arrow key
- ↑ : Up arrow key
- ↓ : Down arrow key
- ⌥ : Option or Alt key
- ↩ : Return or Enter key
- ⇧ : Shift key

| Command | macOS |
| ------- | -------- |
| Decrease Pane Size | ⌘-⌥-- |
| Increase Pane Size | ⌘-⌥-⇧-0 |
